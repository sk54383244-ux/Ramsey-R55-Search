# Ramsey R(5,5) Lower Bound Search

본 프로젝트는 파이썬(Python) 기반의 국소 탐색(Local Search) 및 '지뢰 제거(Local Correction)' 알고리즘을 활용하여, 수학적 난제인 **램지 수 R(5,5)의 하한(Lower Bound)**을 탐색하는 연구입니다. 

## 연구 개요
* **목표**: 램지 수 R(5,5)의 계산적 하한값을 입증하고, 이를 만족하는 그래프 구조를 탐색함.
* **핵심 성과**: 43개의 정점을 가진 그래프에서 5-클릭(Clique)과 5-독립집합(Independent Set)이 존재하지 않는 구조를 성공적으로 도출함.
* **알고리즘**: 그래프 내의 불량 구조(위반 사항)를 감지하고, 국소적 간선 반전을 통해 최적화된 상태로 정화하는 휴리스틱 방식을 적용함.

## 연구 의의
본 결과는 R(5,5) > 43임을 뒷받침하는 계산적 증거로, 조합론적 난제 해결을 위한 알고리즘적 접근의 유효성을 입증함.
